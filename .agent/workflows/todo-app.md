## コード変更後の自動プッシュ手順

// turbo
1. 変更をステージング
```bash
git add .
```

2. コミット（メッセージは変更内容に応じて設定）
```bash
git commit -m "変更内容の説明"
```

3. GitHubにプッシュ
```bash
git push
```
