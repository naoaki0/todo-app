# Todo App

## 🎯 プロジェクトの目標

このアプリは以下の2つの明確な目標のもとに開発されています：

1. **ユーザーに最速で全タスクを完了させる**
2. **ユーザーをこのtodoアプリのタスク完了に依存させる**

### 設計原則

- すべての機能は上記目標に貢献すること
- UIは「タスク完了」という行動を最優先に設計
- 心理的トリガー（ストリーク、通知、ランク等）を活用
- だらだらスクロールよりもタスク完了を魅力的にする

### 主要機能

- **ストリーク機能**: 連続タスク完了日数を記録し、途切れることへの恐怖を利用
- **プッシュ通知**: 14:30/18:00/21:00に通知（深夜型生活パターン対応）
- **成長グラフ**: 視覚的に進捗を確認、ランク付けでモチベーション維持
- **コンボ演出**: 連続完了で派手なエフェクト
- **Firebase同期**: スマホ・PC間でデータ同期

## 技術スタック

- **フロントエンド**: React (CDN)
- **バックエンド**: Firebase (Firestore, FCM)
- **通知**: GitHub Actions + FCM
- **デプロイ**: GitHub Pages
